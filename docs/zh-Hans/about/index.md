# 关于我们

## 关于 SJMC

上海交通大学 Minecraft 社（SJMC）是由一群热爱 Minecraft 及其衍生内容的交大学生们组成的文化艺术类社团，成立于 2021 年 5 月。

SJMC 以 Minecraft 为载体，依托虚拟现实技术塑造数字化时代的交大文化体验。社团以“方块交大”虚拟校园项目为核心，构建高拟真、强沉浸、富交互的三维数字校园，举办云毕业典礼、云学术顶会，设计数字录取通知书、数字孪生文创，并总结实践经验发表多篇论文，产生了广泛影响力。
