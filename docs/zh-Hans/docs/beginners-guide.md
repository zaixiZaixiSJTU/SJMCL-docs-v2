# 新手教程

这是一个提供给 **新手玩家或第一次接触 Minecraft 启动器的用户** 的指南，
旨在用最简单的方式帮助你快速上手 **SJMCL 启动器** 和 Minecraft 游戏。

如果你只想 **尽快开始游戏**，建议先阅读：

* [如何登录账户](#如何登录账户)
* [如何下载游戏](#如何下载游戏)

如果需要深入了解某个主题，可以点击文中提供的链接查看详细文档。

# 如何登录账户

你需要使用 **Microsoft 账户** 购买游戏才能游玩 Minecraft，或者使用 **离线账户 / 外置账户**。

你可以在 [Minecraft 官方网站](https://www.minecraft.net/) 或 [Xbox 商店](https://www.xbox.com/) 购买游戏。

完成购买后，在 **SJMCL** 上方点击 **账户**，然后点击右上角的 **添加角色**，即可创建 **离线角色**，或登录 **微软角色**、**第三方认证角色**。

你也可以从其他启动器 [导入账户](#如何导入账户)。

---

# 如何导入账户

SJMCL 支持从其他 Minecraft 启动器导入已有账户。

操作步骤如下：

1. 在 **SJMCL** 上方点击 **账户**
2. 点击左下角的 **从其他启动器导入**
3. 启动器会自动检测并列出其他启动器中的角色
4. 点击角色头像左侧的 **选择框**，选择需要导入的角色
5. 确认选择后，点击右下角的 **导入** 按钮

完成后，所选角色就会被导入到 SJMCL 中，你可以直接使用这些账户启动游戏。

# 如何下载游戏

使用 **SJMCL** 下载 Minecraft 非常简单，只需几个步骤即可完成。

如果你 **还没有安装任何游戏版本**，可以按照下面步骤操作：

1. 在启动界面找到并点击上方的 **实例**，进入实例管理界面。
   ![启动界面](/images/beginners-guide-launch.png)

2. 点击左下角的 **添加与导入** 按钮。
   ![实例界面](/images/beginners-guide-games.png)

3. 在弹出的界面中选择 **安装新实例** 选项卡。
   ![添加与导入界面](/images/beginners-guide-addAndImport.png)

在这里会列出所有可用的 Minecraft 游戏版本（版本号通常 **从新到旧排序**）。
选择一个你想游玩的版本，我们通常推荐 **最新的正式版**，随后点击 **下一步**。

![选择游戏版本](/images/beginners-guide-gameClient.png)

接下来你会看到 **模组加载器选择界面**。
这里可以为游戏安装模组加载器，例如 **Forge、Fabric** 等。

![选择模组加载器](/images/beginners-guide-gameLoader.png)

> 如果你只是想体验 **原版 Minecraft**，可以直接点击 **下一步** 跳过此步骤。

> 什么是模组加载器？应该选择哪一个？
> 请参考：**【安装模组加载器】**（暂未编写）

随后进入 **实例设置界面**。

在这里你可以设置：

* 实例名称
* 实例描述
* 实例图标
* 游戏存储位置

![实例设置](/images/beginners-guide-basicSettings.png)

设置完成后，点击右下角的 **完成** 按钮。
启动器会自动跳转到下载页面，并开始下载你选择的游戏版本。

下载完成后：

1. 点击上方的 **启动** 返回启动页面
2. 点击右下角的 **启动游戏**

现在，你就可以进入 Minecraft，开始你的创造之旅了！

# 什么是资源包

你可能听到有其他玩家讨论好看的「材质包」。

「材质包」通常是指修改游戏纹理的 **资源包**，也是 Minecraft **1.5 及更早版本**使用的旧称呼。

「材质」和「材质包」实际上是早期的误译，目前官方和社区普遍使用的译名是 **纹理** 和 **纹理包**。因此，如果你看到有人提到「材质包」，通常指的就是 **资源包中的纹理部分**。

**资源包（Resource Pack）** 是一种可以修改游戏资源的文件。
它可以改变游戏中的多种内容，例如：

* 纹理（贴图）
* 模型
* 音乐
* 音效
* 语言文件
* 用户界面

通过安装资源包，你可以改变 Minecraft 的整体视觉和听觉体验。一个优秀的资源包甚至可以让游戏呈现出完全不同的风格。

**如何使用资源包？**
查看：[加载资源包教程](https://zh.minecraft.wiki/w/Tutorial:%E5%8A%A0%E8%BD%BD%E8%B5%84%E6%BA%90%E5%8C%85)

**想制作自己的资源包？**
深入了解：[资源包详细资料](https://zh.minecraft.wiki/w/%E8%B5%84%E6%BA%90%E5%8C%85)

**获取资源包**

你可以在以下网站获取资源包：

# 什么是模组（Mod）

你可能听说过我的世界拥有丰富的模组（Mod）资源。模组是一类能改变或增加游戏内容的强大工具，它允许你新增方块、生物、物品与机制等内容。通过安装模组，你可以获得更加丰富的游戏内容、更便捷的玩法体验，甚至让游戏呈现出完全不同的风格。

**获取模组**

你可以在以下网站获取模组信息，并在 **SJMCL** 中下载和安装：

* [MC 百科](https://www.mcmod.cn/)
* [CurseForge](https://www.curseforge.com/minecraft)
* [Modrinth](https://modrinth.com/)

# 安装 Mod 后游戏报错怎么办

安装模组后游戏无法启动是比较常见的问题。

通常原因包括：

* 模组版本不兼容
* 缺少前置模组
* 模组之间冲突

解决方法通常包括：

1. 检查模组版本是否与游戏版本匹配
2. 确认是否安装了前置模组
3. 删除最近安装的模组

# 如何选择游戏的 Java 版本

**SJMCL** 会自动管理 Java，并根据不同版本的 Minecraft 自动选择合适的 Java 版本。

当启动游戏时，如果没有检测到合适的 Java，启动器会提示你下载所需的 Java 版本，点击 **下载** 即可自动完成安装。对于新手玩家来说，**保持默认设置即可正常游玩**。

如果你确实需要手动安装或指定 Java 版本，可以按照下面步骤操作。

---

**安装 Java**

1. 进入 **设置 → Java**，这里会显示启动器已安装的 Java 或系统中已检测到的 Java。

2. 如果需要下载新的 Java，点击 **下载 Java**，选择需要的版本，然后点击 **确定**。

3. 如果你的电脑已经安装了 Java，但启动器没有检测到，可以点击 **添加 Java**，然后选择 Java 安装目录中的可执行文件：

   * Windows：选择 `java.exe`
   * Linux / macOS：选择 `java`

添加完成后，该 Java 就会出现在 Java 列表中。

---

**选择 Java**

你可以为所有实例使用同一个 Java，也可以为某个实例单独指定 Java。

**为所有实例设置 Java：**

进入 **设置 → 全局游戏设置**。
默认情况下，启动器会 **自动选择合适的 Java**。如果需要，你也可以取消自动选择，并手动指定要使用的 Java 版本。

该 Java 将用于所有使用全局设置的实例（一般不推荐新手修改）。

**为单个实例设置 Java：**

1. 打开 **实例** 页面
2. 进入该实例的 **设置**
3. 开启 **启用特定游戏设置**
4. 选择需要使用的 **Java 版本**

这样设置的 Java 只会用于该实例的启动。

> ⚠ **重要提示**
>
> 如果在更换 Java 后出现游戏崩溃或无法启动的问题，请立即切换回启动器自动选择的默认 Java 版本。
