# 安装模组加载器

模组加载器是一种软件工具，允许你在 Minecraft 中安装和运行模组（Mod）。不同的加载器有不同的特点和兼容性。

当你在安装新的游戏客户端时，会看到其中有 `Forge`、`Fabric`、`NeoForge`、`Optifine` 这 4 个 Mod 加载器可供选择。

当选择 Fabric 模组时，会默认附带 `Fabric API`，并可以在界面下方自行取消勾选。

如果不确定选择什么模组加载器

推荐在 1.14+ 版本选择 `Fabric` / `NeoForge` 加载器，在 1.14 以下版本选择 `Forge` 加载器。
