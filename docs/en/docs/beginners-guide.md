# Beginner Guide

This guide is intended for **new players or users who are using a Minecraft launcher for the first time**.
It aims to help you quickly get started with **SJMCL Launcher** and Minecraft in the simplest way.

If you just want to **start playing as soon as possible**, we recommend reading:

* [How to Log in to an Account](#how-to-log-in-to-an-account)
* [How to Download the Game](#how-to-download-the-game)

If you want to learn more about a specific topic, you can follow the links provided in this document.

---

# How to Log in to an Account

You need a **Microsoft account** that owns Minecraft in order to play the game, or you can use an **offline account / external authentication account**.

You can purchase the game from the [Minecraft Official Website](https://www.minecraft.net/) or the [Xbox Store](https://www.xbox.com/).

After purchasing the game, click **Accounts** at the top of **SJMCL**, then click **Add Profile** in the upper-right corner.
From there you can create an **Offline Profile**, or log in with a **Microsoft Account** or **Third-party Authentication Account**.

You can also [import accounts](#how-to-import-accounts) from other launchers.

---

# How to Import Accounts

SJMCL supports importing existing accounts from other Minecraft launchers.

Follow these steps:

1. Click **Accounts** at the top of **SJMCL**
2. Click **Import from Other Launchers** in the lower-left corner
3. The launcher will automatically detect and list profiles from other launchers
4. Select the profiles you want to import using the checkbox next to the avatar
5. After confirming your selection, click the **Import** button in the lower-right corner

Once completed, the selected profiles will be imported into SJMCL and can be used to launch the game.

# How to Download the Game

Downloading Minecraft with **SJMCL** is very simple and only takes a few steps.

If you **have not installed any game versions yet**, follow the steps below:

1. On the launcher home page, click **Instances** at the top to open the instance management page.
   ![Launcher Page](/images/beginners-guide-launch.png)

2. Click the **Add & Import** button in the lower-left corner.
   ![Instances Page](/images/beginners-guide-games.png)

3. In the popup window, select the **Install New Instance** tab.
   ![Add and Import Page](/images/beginners-guide-addAndImport.png)

Here you will see a list of all available Minecraft versions (usually **sorted from newest to oldest**).
Choose a version you want to play. For most players, we recommend installing the **latest release**, then click **Next**.

![Select Game Version](/images/beginners-guide-gameClient.png)

Next, you will see the **mod loader selection page**.
Here you can install mod loaders such as **Forge** or **Fabric**.

![Select Mod Loader](/images/beginners-guide-gameLoader.png)

> If you only want to play **vanilla Minecraft**, you can simply click **Next** to skip this step.

> What is a mod loader? Which one should I choose?
> See: **Installing Mod Loaders** (not written yet)

After that, you will enter the **instance settings page**.

Here you can configure:

* Instance name
* Instance description
* Instance icon
* Game storage location

![Instance Settings](/images/beginners-guide-basicSettings.png)

After finishing the settings, click the **Finish** button in the lower-right corner.
The launcher will switch to the download page and automatically download the selected game version.

After the download is completed:

1. Click **Launch** at the top to return to the launch page
2. Click **Launch Game** in the lower-right corner

Now you can enter Minecraft and start your adventure!

While waiting for the game to download, you might want to learn about its core gameplay.

# What Is a Resource Pack

You may have heard other players talking about good-looking “texture packs”.

“Texture pack” usually refers to a **resource pack that modifies game textures**, and it was the name used before Minecraft **1.5**.

The terms “texture” and “texture pack” were early translations. Today, the commonly used term is **resource pack**. When people mention texture packs, they usually mean the texture part of a resource pack.

A **Resource Pack** is a collection of files that can modify various game assets, such as:

* Textures
* Models
* Music
* Sound effects
* Language files
* User interface

By installing resource packs, you can dramatically change the visual and audio experience of Minecraft.

# What Is a Mod

Minecraft has a huge ecosystem of **mods (modifications)**.

Mods are programs that **change or extend the game’s content**. By installing mods, you can add new content such as:

* Blocks
* Creatures
* Items
* Mechanics
* Dimensions
* Gameplay systems

With mods, you can greatly expand the gameplay experience and even transform Minecraft into a completely different game.

**Where to get mods**

# What Should I Do If the Game Crashes After Installing Mods

Game crashes after installing mods are fairly common.

Typical causes include:

* Incompatible mod versions
* Missing dependency mods
* Conflicts between mods

Common solutions include:

1. Check whether the mod version matches the Minecraft version
2. Make sure required dependency mods are installed
3. Remove recently installed mods

# How to Choose the Java Version

**SJMCL** automatically manages Java and selects the appropriate Java version for different Minecraft versions.

When launching the game, if no suitable Java installation is detected, the launcher will prompt you to download the required Java version. Simply click **Download** to install it automatically. For most users, **the default settings are sufficient**.

If you need to manually install or select a Java version, follow these steps.

---

**Installing Java**

1. Go to **Settings → Java**, where you will see Java versions installed by the launcher or detected on your system.

2. To download a new Java version, click **Download Java**, select the required version, and click **Confirm**.

3. If Java is already installed on your system but not detected, click **Add Java** and select the Java executable file:

   * Windows: `java.exe`
   * Linux / macOS: `java`

After adding it, the Java installation will appear in the Java list.

---

**Selecting Java**

You can either use the same Java for all instances or specify a Java version for a particular instance.

**Set Java for all instances:**

Go to **Settings → Global Game Settings**.
By default, the launcher **automatically selects an appropriate Java version**, but you can disable this option and manually select one if necessary.

This Java version will be used by all instances using global settings (not recommended for beginners).

**Set Java for a single instance:**

1. Open the **Instances** page
2. Enter the instance **Settings**
3. Enable **Instance-specific settings**
4. Select the desired **Java version**

This Java version will only be used when launching that specific instance.

> ⚠ **Important**
>
> If the game crashes or fails to launch after changing the Java version, switch back to the automatically selected default Java version immediately.
